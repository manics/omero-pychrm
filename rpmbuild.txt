Instructions for creating OMERO PyCHRM RPMs
===========================================


OMERO PyCHRM
------------
1. Clone the git repository, and create a source archive:
	git clone git@github.com:manics/omero-pychrm.git
	cd omero-pychrm
	git checkout cleanup_10583
	VERSION=0.1.0
	git archive -o omero-pychrm-$VERSION.tar.gz --prefix \
		omero-pychrm-$VERSION/ cleanup_10583
2. Move omero-pychrm-$VERSION.tar.gz to the rpmbuild/SOURCES directory.
3. Update the RPM spec file if necessary: rpm/omero-pychrm.spec. Note that
   this is a heavily modified version of the spec file generated by:
	python setup.py bdist_rpm --spec-only
4. Create the source archive:
	rpmbuild -bs rpm/omero-pychrm.spec
5. Create the binary RPMs, either using rpmbuild or (ideally) using mock.


PyCHRM
------
1. Create a source archive. Note at present omero-pychrm only works with
   an old version of PyCHRM (SVN revision 558) with some additional
   patches. This should be fixed at some point.
	VERSION=0.1.0
	svn export -r 558 http://wnd-charm.googlecode.com/svn/pychrm/trunk \
		pychrm-$VERSION
	tar -zcvf pychrm-$VERSION.tar.gz pychrm-$VERSION
2. Move pychrm-$VERSION.tar.gz to the rpmbuild/SOURCES directory.
   Copy rpm/pychrm-r558-demo_feb_2013.patch to the rpmbuild/SOURCES directory.
3. Update the RPM spec file if necessary: rpm/pychrm.spec. Note that
   this is a heavily modified version of the spec file generated by:
	python setup.py bdist_rpm --spec-only
4. Create the source archive:
	rpmbuild -bs rpm/omero-pychrm.spec
5. Create the binary RPMs, either using rpmbuild or (ideally) using mock.
